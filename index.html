Tentang Saya

Saya adalah seorang frontend developer pemula yang antusias dan terus belajar secara mandiri untuk menciptakan antarmuka web yang bersih, responsif, dan fungsional.

Berbekal keterampilan dalam HTML, CSS, dan JavaScript dasar, saya telah mengembangkan beberapa proyek pribadi untuk mengasah kemampuan saya dalam membangun website dari awal secara efisien.

Saya percaya bahwa kombinasi desain sederhana dan struktur kode yang rapi adalah kunci dari pengalaman pengguna yang baik. Saat ini, saya aktif memperluas pengetahuan saya melalui kursus online dan eksplorasi proyek open source.
